Professional Grooming Services

Site Name

A.J. Cutz Online BarberShop

This name represents a professional barbershop brand that focuses on high-quality grooming services. The name "A.J. Cutz" is unique, memorable, and reflects modern barber culture while maintaining a professional identity.

Optional domain availability: ajcutzbarbershop.com

Site Purpose

The purpose of this website os to provide an online presence ofr A.J. Cutz Online BarberShop by informing visitors about available grooming services, pricing, and contact information. The site will also demonstrate responsive design, accessibility, and dynamic functionality through using HTML, CSS, and JavaScript.

Scenarios

  • What service does A.J. Cutz offer, and how much does each service cost?
  • How can I book an appointment or contact the barber?
  • Where is A.J. Cutz located, and what are the business hours?

Color Schema

The following color scheme will be used throughout the site:

  • Black (#000000): Heading, navigation bar footer
  • Gold (#C9A24D): Buttons, accents, highlights
  • White (#FFFFFF): Background and body text areas

Wireframe

Mobile View

- Logo and site name at the top
- Hamburger navigation menu
- Main content stacked vertically
- Call-to-action button below introduction
- Footer at the bottom

Desktop View

- Logo and site name aligned left
- Horizontal navigation menu
- Main content displayed in sections
- Featured image beside introduction text
- Footer across the bottom of the page

Planned JavaScript Features

  • Multiple JavaScript functions for dynamic content
  • DOM selection, modification, and event handling
  • Conditional logic for validation and interaction
  • Arrays and objects to store service data
  • Template literals for dynamic HTML output
  • localStorage to remember user preferences

Accessibility & Responsiveness

The site will use semantic HTML, proper heading structure, alt text for images, sufficient color contrast, and responsive layouts for mobile and desktop views.
